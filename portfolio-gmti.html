GMTI STIKOM BALI

Website for new students undergoing orientation. Students can obtain information about ITB STIKOM BALI and submit assignments given by the committee.

In this case, I am assigned as a backend developer to create APIs for the front end, which uses Vue JS. Here, users or students can view information about UKM (Unit Kegiatan Mahasiswa), announcements, activity schedules, assignment submissions, and see group members and committee members.

Tech

Laravel MySQL Rest API
