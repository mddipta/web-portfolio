HIPMI EXPO DENPASAR

HIPMI Expo is an event organized by HIPMI (Himpunan Pengusaha Muda Indonesia) Denpasar. The event is held at ITB STIKOM BALI and features various seminars and product exhibitions.

In this case, I am developing features for login use OTP and the OTP send to WhatsApp number, displaying events from the admin, and implementing a discount voucher feature. Additionally, I am creating an admin page for attendance tracking using QR Code scanning from each user's page, creating vouchers, displaying transactions, and managing affiliate transactions.
