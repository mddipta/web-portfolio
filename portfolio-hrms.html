LinovHR

LinovHR is a comprehensive Human Resources (HR) software solution designed for businesses in Indonesia. It offers a centralized platform to streamline various HR processes, from hiring to retirement. The software includes modules for personnel administration, time management, payroll, recruitment, employee self-service, performance management, learning management, and more. LinovHR aims to enhance business efficiency and ensure secure data handling for HR departments across various industries. Additionally, the platform provides analytics and a document management system to optimize HR operations.

Here, I work as a full stack developer. In this application, there are several features that I have created. On the front end, I developed the selection process for each company, with different selection stages for each company. I have two versions for this: the first is a Kanban board and the second is a table. Then, I created the front end for adding candidates.

On the back end, I developed features such as the question bank, job vacancy, and data migration using Excel, following the existing template.
